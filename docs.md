# Ring
## Methods:
Let $R$ denote a ring.

```contains(a)```

Arguments:
* a -- an object.

Returns:
* True if $a \in R$. False otherwise.

```add(a, b)```

Arguments:
* a -- an element of $R$.
* b -- an element of $R$.

Returns:
* a+b -- the sum of $a$ and $b$ in R.

```sub(a, b)```

Arguments:
* a -- an element of $R$.
* b -- an element of $R$.

Returns:
* a-b -- the difference of $a$ and $b$ in R.

  ```mult(a, b)```

Arguments:
* a -- an element of $R$.
* b -- an element of $R$.

Returns:
* a*b -- the difference of $a$ and $b$ in R.

```to_str(a)```

Arguments:
* a -- an element of $R$.

Returns:
* a string representation of the element $a \in R$
